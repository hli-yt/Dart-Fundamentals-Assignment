# Dart Fundamentals Assignment

## Overview
This project contains two Dart applications that analyzes a list of fundamental programming concepts. It demonstrates the use of loops, conditioal functions and list of operations in Dart.

## Task 1: Number Analysis App
- Implements list analysis functions using loops and conditionals
- Features: sum, average, min/max, count negative numbers and sort in ascending order 

## Task 2: Async Calculator App
- OOP Calculator class with async/await
- Exception handling for division by zero
- Calculation history tracking
